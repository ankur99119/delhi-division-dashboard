🚆 First-Time Setup

This dashboard reads live data from your Google Sheet. Complete setup once.

1
Open your Google Sheet → Extensions → Apps Script → paste the AppsScript.gs file → Save → Run setupAll()
2
Make sheet public: Share → Anyone with link → Viewer → Done
3
Deploy as Web App: Deploy → New Deployment → Web App → Execute as: Me → Anyone → Deploy → Copy URL
4
Paste your Web App URL below (needed to save data from dashboard):
